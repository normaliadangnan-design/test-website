PUBLIC AZURA 🔐
Simple login & register system na may dashboard categories. Gawa sa HTML, CSS, at JavaScript lang.

🚀 Features
Register & Login - Gumawa ng account gamit username + password
LocalStorage - Naka-save yung users sa browser mo
Dashboard - Pagka-login may main title na "PUBLIC AZURA"
Categories - 3 sections: GOD$, ABOVE ALL, at STAFF
Logout - Balik sa login screen
💻 How to Use
Open mo lang yung index.html sa browser
Mag-register ka muna ng account
Login gamit yung username/password na ginawa mo
Makikita mo na yung dashboard with categories
⚠️ Note
Pang-demo lang 'to. Naka-localStorage lang yung data kaya nasa browser lang. Sa real app dapat naka-backend + database na at naka-hash yung passwords.

🛠️ Tech Stack
HTML5
CSS3
Vanilla JavaScript
